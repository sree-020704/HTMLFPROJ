What is Html?

Hypertext Markup Language

HyperText: Refers to the system of links that connect web pages to one another, allowing users to navigate between different resources on the internet.

Markup Language: Means that HTML uses special codes, known as tags, to annotate text and define its structure and presentation. It is not a programming language, but rather a descriptive one.
